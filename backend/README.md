# Fast-Serve-Backend

## Description
This is the backend repository for fast-service project

## Prerequisites
- node v20
- mongodb instance

## Configuration
1. **Env files**
   ```plaintext
    PORT=
    MONGO_URL=
    MONGO_DB=
   ```

## Running Application
- install dependencies using `npm install`
- start the server using `npm start` or using `npm run dev`

## Running test
- run te test using `npm run test:unit`
